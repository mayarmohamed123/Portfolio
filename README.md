# Gerold - Web Developer + UX Designer Portfolio

This is a professional personal portfolio website for **Gerold**, a Web Developer and UX Designer. The project showcases skills, recent works, and a contact section for potential collaborations.

## 🚀 Live Demo

Check out the live version here: [Portfolio Demo](https://portfolio-three-beta-faziobimgk.vercel.app/)

## ✨ Features

- **Personal Branding**: Clean and modern introduction with a downloadable CV option.
- **Projects Showcase**: Display of recent works including UI/UX, Branding, and App projects.
- **Skill Progress**: Visual representation of proficiency in tools like Figma, React, and JavaScript.
- **Testimonials**: Section for client feedback.
- **Contact Form**: Integrated form for service inquiries and direct contact information.
- **Responsive Design**: Fully optimized for various screen sizes (Mobile, Tablet, Desktop).

## 🛠️ Tech Stack

- **HTML5**: Semantic structure.
- **CSS3**: Custom styling, animations, and responsive layouts.
- **Google Fonts**: Sora for modern typography.
- **Font Awesome**: High-quality icons for social links and interface elements.
